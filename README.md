# Catering Personnel Telegram Bot

Telegram-бот для автоматизации управления персоналом кейтеринговой компании.

## Функции

- регистрация сотрудников;
- создание мероприятий;
- добавление смен;
- запись персонала на смены;
- контроль свободных мест;
- уведомления сотрудников;
- формирование списков персонала;
- экспорт данных в Word и Excel;
- работа с базой данных MySQL.

## Технологии

- Python
- Aiogram
- MySQL
- OpenPyXL
- python-docx
